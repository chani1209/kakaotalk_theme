# iOS 카카오톡 테마 크리에이터 (KakaoTalk Theme Creator for iOS)

iOS용 카카오톡 테마를 쉽고 빠르게 제작할 수 있는 웹 기반 도구입니다. 복잡한 코딩 없이 클릭만으로 나만의 테마를 만들어보세요.

## 주요 기능 (Features)

- **실시간 미리보기**: 변경 사항을 메인 화면, 채팅방, 잠금화면, 탭바 등 다양한 화면에서 실시간으로 확인할 수 있습니다.
- **다양한 기기 지원**: iPhone SE부터 15 Pro Max까지, 그리고 Android 기기의 화면 비율에 맞춰 미리보기 크기가 자동으로 조정됩니다.
- **이미지 에디터**: Filerobot 이미지 에디터가 내장되어 있어, 아이콘이나 배경화면을 정확한 규격(예: 162x162)에 맞게 자르고 편집할 수 있습니다.
- **모듈형 구조**: 유지보수가 쉽고 확장 가능한 모듈형 JavaScript Codebase입니다.
- **프리미엄 UI**: 작업에 집중할 수 있는 깔끔하고 직관적인 Light Mode 디자인을 제공합니다.

## 파일 구조 (File Structure)

```
kakaotalk_ios/
├── index.html         # 메인 애플리케이션
├── js/
│   ├── config.js      # 설정 및 상수 정의 (기기 프리셋 등)
│   ├── state.js       # 상태 관리 (프로젝트 데이터)
│   ├── editor.js      # 이미지 에디터 로직
│   ├── preview.js     # 실시간 미리보기 업데이트
│   ├── css-generator.js # CSS 파일 생성 로직
│   ├── build.js       # .ktheme 파일 빌드 및 내보내기
│   └── app.js         # 앱 초기화 및 이벤트 핸들러
├── sample_theme/      # 샘플 테마 리소스 (Apeach)
├── KakaoTalk_Theme_Guide_iOS.pdf # 공식 제작 가이드
└── README.md
```

## 사용 방법 (How to Use)

1. 웹 브라우저(Chrome, Edge 등)에서 `index.html` 파일을 엽니다.
2. **기본 설정**: 
   - **Step 1**에서 테마 이름, 버전, 작성자 등 기본 정보를 입력합니다.
3. **디자인 및 편집**: 
   - **Step 2~4**를 순서대로 진행하며 색상을 선택하고 이미지를 업로드합니다.
   - 우측 **미리보기 패널**의 드롭다운을 통해 다양한 기기 화면에서 어떻게 보이는지 확인할 수 있습니다.
   - **샘플 로드**: '샘플 불러오기' 버튼을 클릭하면 기본 어피치 테마와 이미지가 적용되어 빠르게 테스트해볼 수 있습니다.
4. **테마 생성**:
   - 모든 작업이 끝나면 우측 상단의 **테마 생성 (Build)** 버튼을 클릭합니다.
   - 다운로드된 `.ktheme` 파일을 아이폰으로 전송한 뒤 카카오톡에서 열어 적용합니다.

## 개발 정보 (Development)

이 프로젝트는 별도의 번들링이나 빌드 과정 없이 **Vanilla JavaScript (ES6+)**와 **Tailwind CSS (CDN)**로 작성되었습니다. `index.html`을 수정하여 바로 기능을 확장할 수 있습니다.

- **제작 가이드**: [카카오톡 iOS 테마 제작 가이드 v25.11.0](KakaoTalk_Theme_Guide_iOS.pdf)를 준수합니다.
